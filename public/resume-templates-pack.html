About these templates

  1. Jake's Resume — Most-forked CS resume on GitHub. Single column, Latin Modern Roman, small-caps section headings under a rule. github.com/jakegut/resume
  2. Deedy — 1/3 + 2/3 asymmetric two-column layout in Lato/Raleway. Sidebar holds Education, Coursework, Skills; main column holds Experience & Projects. github.com/deedy/Deedy-Resume
  3. Awesome-CV by posquit0 — Source Sans Pro, "awesome-blue" accent (#0E5484), tagline/quote slot, iconified contact row, structured experience entries. github.com/posquit0/Awesome-CV
  4. sb2nov Resume — Classic serif single-column. The parent template that Jake's was forked from; still in active use. github.com/sb2nov/resume
  5. RenderCV — Classic Theme — YAML → PDF builder for engineers/academics. Centered header + partial rule lines beside section titles. github.com/rendercv/rendercv
  6. Harvard Classic (HBS bullet-point format) — Times New Roman, pure black-and-white, all-caps name with letter-spacing. The most universally ATS-safe layout — recommended by Harvard FAS Career Services. careerservices.fas.harvard.edu

How to open

Run npm run dev in resuscore/ and visit http://localhost:3000/resume-templates-pack.html — or just double-click the file in resuscore/public/ to open it in any browser.

Wiring into JobScorer

To use these as real PDF templates inside the app, each one needs a @react-pdf/renderer port under resuscore/src/components/ResumeRenderer/ (similar to your existing ClassicPdfDocument, LondonPdfDocument, etc.), then registered in resuscore/src/templates/catalog.ts.
